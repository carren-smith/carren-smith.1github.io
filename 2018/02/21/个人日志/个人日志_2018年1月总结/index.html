【工作】 7分

  1. 分析报告：对于年度报告的提纲，其实并没有想清楚；只是有的大体的方向，具体怎么写并没有弄明白；例如交易盈亏，你要表达的是什么，然后才是决定你的展现形式； 对于PPT制作，自己会做这样的动作，但到了分析报告，则没有很好地把这个原则贯彻下去。对于提纲，对于内容的展开，的确没有想清楚。下次，要细化到具体的分析维度。 例如，对于开户，不仅仅是内容，而是要表什么关于开户的什么内容，到这一维度。

    另外，发现用PPT写东西节奏都会变得很慢，不知道是效率问题还是什么。几页片子花了整整一天还没有搞定，弄到23:00才从公司出啦。

    【启发】对于分析，思考的程度一定要细化到具体分析的维度，否则只能说明其实你并没有把这个问题考虑清楚。

    【启发】若最终要用PPT做呈现，内容可以先用Word完成，最后在进行内容的转移与格式的调整；因为很明显，在PPT使用的过程中，无形中生产的效率低了下来。就发现这个现象而言，就足够你去改变你的行为。

  2. 呈现：温习《图形的表达方式》，还是很有帮助的；核心：先确定你要表达的信息，再选择图表，并将要表达的内容作为标题。 一次/一个图 只说一件事；不要混着说；一定要充分认识到文字是线性的； 这些内容，在之后帮助Maolt 共同完成 LCS年度总结报告的时候 从我的口中说了出来。之前都是自己应用，这一次从应用者转变为讲授者，这是一个新的体验。

  3. 数据表单 - 特征：数据表要根据特征来划分；动态表，是每天一条记录；静态表是固定不变的；若把两者混在一起，当数据出现问题的时候，你往往很难找到原因。 —— Zhengyj 2018.01.16 19:30

    【启发】特征不同的表单，分开放置，不好混在一起。因为若把两者混在一起，当数据出现问题的时候，你往往很难找到原因。

  4. 数据核对：核对数据，关键问题是统一口径；然后就是算法的问题；收入的数据，一定要有强烈的保密意识；文档加密，发送；—— Zhengyj

  5. 产品会议：参加产品会议；沟通了新功能的上线；对于数据部门，我们要关心的是新功能过程中，能够产生哪些数据，我们能以什么样的方式去获得，并且能够与之前的数据进行区分。这个才是作为数据人员最需要关注的。

    【启发】必须要认识到整个功能，它背后的真正的目的是什么。为了引流，还是为了某一个功能的实现。只有思考到这个层次，才能真正知道应该关注什么数据，才能用好数据。

【精要之事】 8分

​ 自从将精要之事细化到可执行的行为之后，就尝试开始执行。刚开始的几天还是有所意识，但中间有几天退化到完全没有在意；但通过再次回顾自己的思考与行动之后，运用起了强制唤起的方法，还真的有所帮助，加强了意识。

< 关于预留时间 >

  1. 预留时间，最佳解决措施就是09:00之前到公司，然后花15分钟来思考。

< 关于保护时间 >

  1. 做一件事的期间，应该不主动去看即时消息；有这个意识，但并没有做好；致使经常在多个任务之间切换；
  2. 处理一件事情的时候，已经尽量不去主动分心；
  3. 贴上 “保护预留时间” 的便签；在板夹白纸中贴上 “觉察点” 的便签；
  4. 早上完成的事项 （任务清单 - 成功清单 - SCQA-K&R ），后面的保护预留时间，没有走完

< 关于觉察点 >

  1. 在工作开始前，会思考接下来要做的事情是什么；不仅仅是早上的时候，还有下午开始的时候；

  2. 中途被打断的时候，或者完成任务的时候，并没有意识到觉察点；首先你要意识到，你被打断了。

  3. 在板夹上贴上了 触发点 的便利贴；强制唤起还是很有用的；

  4. 虽然有好几次都意识到了觉察点，但第一没有记录下来，第二没有好好珍惜；

  5. 设计了记录意识到觉察点的表格

  6. 只要意识到觉察点，就记录在OnoNote，无论之后是否有去思考。

  7. 晨间日志中回顾觉察点的两个场景

  8. 在左侧书籍的位置贴上 “意识到觉察点 - 当前场景”

  9. 在抽纸一侧的位置贴上 “你现在做的是当前最重要的事吗？”

    开始在 左侧书籍 和抽纸一侧 写上了便利贴；虽然效果并不是很好，但偶尔一瞬间，只要还是能让你把意识从当下拉出来。让你能够作为一个旁观者观察自己的思绪与状态，那就足够了。

< 关于重要的事 >

  1. 早上的时候，一定要先去做最重要的事，不要被其他事所分心；一大早被一个主任弄得很心烦，说什么报表的问题。搞得一大早不仅心情弄得糟糕，而且效率也非常低；
  2. 时常问自己，“ 为了达成XXX，当前最重要的一件事是什么？ “ 。这个能让你对即将做的事有一个清晰的衡量标准；即明确了你的目的是什么，同时让你把精力放到关键问题上来。

【自我思索】 9分

  1. 关系：把自己的孤寂感都告诉了Lijj，想找她倾诉，但她有自己的心境，她不想出门就待在了宿舍；其实，在这段友情里面，很明显我们是不对等的；对于如何做自己，会迷茫；不知道应该怎么样。其实或者不该直接这样说，而是等约出来的时候在跟她说。其实，都一样；如果对方愿意花时间的话，直接你的苦恼跟对方说，对方就会有回应的；不需要那么多的招式和套路。

  2. 环境：对于环境对自己的影响，很重要的；不要高估自己的意志力；在公司学习，因为没有约到对方，加之寒冷，后面就在看动漫《家庭教师》；相比前两天在 “大众书局” 明显低效了很多；

    周六15:00才出门去大众书局；人的意志力是脆弱的。当我坐在书局里面，开始认真思考与码字的时候，就完全不会觉得外面寒冷的天气。人也不会慵懒了。另外，在公司看书与思考，真的很低效；（如果很的要思考，那就去大众书局）

    【启发】再一次认识这个现实，所以改变行为就是下次要学习就早点出门，以及寻找适合的环境。

  3. 抗干扰：自己的抗干扰能力很差；涉及到需要静心思考的内容，每次都需要等到周末的时候才能好好思考。在办公室，总觉得很难静下心来；先不说白天的时候，即使到了晚上下班的时候，也依旧如此。

    周六图书馆的时候，旁边坐着Yangye 竟然也会对自己产生一些影响。使得自己一开始并不能很好地集中精神。

  4. 睡眠：睡眠不好，会直接影响第二天一整天的精神状态；这个月，太多次这样的情况了。

  5. 规划：《做一个精要的人》；作为2018年要做的一件事，努力完善中；一定要把行为细化到可执行的地步，才能真正去执行和检验。

  6. 记录：当说到彼此第一次见面的日期，Guosx很快从以往的总结里面翻出来；再一次好好看 “成长为自己” 的架构。还是觉得有必要好好弄一弄自己的文章；博客可以好，公众号也好；不需要把所有内容都放上去；

  7. 个人素养：从 Zhengyj 那里拿回了自己的书，但是没有及时跟她说，是我的问题。

  8. 时间：缺少时间；总觉得属于自己的时间很少；

  9. 论思考的重要性：明明一个渠道分和人效的内容，花费了几乎一整天的时间；原因在于动笔之前并没有思考清楚；同事问，你想要表达什么？这个时候，我竟然答不上来。所以，以后一定要先思考清楚，在去行动。—— By 《年度报告》 渠道消费&开发人效

    对分析报告的思路，其实之前并没有弄清楚；即要传达什么内容，只是有了大体的框架，具体的要传达的怎样的内容并不清楚；例如：传达交易盈亏，但是要传达的是什么样的交易盈亏。总体的？趋势的？分组的？

    当天虽然有意识都觉察点，但自己以太忙为理由而把它忽视掉了；其实，正是越忙的时候，越需要思考；
